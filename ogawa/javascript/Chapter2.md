# javascriptの仕事（Twiiter）
    １．データのやり取り
        
    ２．DOM操作・・・Webページの内容（HTML要素）を動的に変更すること。
        DOM(Document Object Model)とは
            HTMLやXMLでマークアップされた文書を、JavaScriptなどのスクリプトからアクセス、操作できるようにするAPIのこと。
        DOM操作の例
            ・ボタンを押したときに色を変える
            ・要素を追加、削除する etc…

# データのやり取り
    例：
        １．無限スクロール
        ２．ローディングマークが表示
        ３．WebAPI(サーバー)に対してAPIをたたきに行く
            ・WebAIP・・・web上でHTTP通信を使ってデータをやり取りするための仕組みのこと
            ・WebAPIを使う理由・・・フロントから直接データベースにアクセスできてしまうと悪意を持って攻撃できてしまうから、WebAPIで認証を挟む必要がある
        ４．JSONでレスポンスを返す
        ５．パースをする(JSONをHTMLに変換する)
            ・パース・・・データを解析して、プログラムが扱える形式に変換すること
