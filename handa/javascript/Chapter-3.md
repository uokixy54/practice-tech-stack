# 1.オブジェクト・プロパティ・メソッドの用語紹介
    オブジェクトとは
        データと機能をまとめたもの
            ・プロパティ（オブジェクト内のデータに相当するもの）
            ・メソッド（オブジェクト内の機能に相当するもの）
    オブジェクトは入れ子構造にすることができる。

# 4.Window,Documentについて紹介
    JavaScriptはブラウザで動く言語。
    ブラウザ自体もオブジェクトで定義されていて、メソッドやプロパティにアクセスできる。
    その機能を使って、データのやり取りやDOM操作を行う。
    ＊サーバーで動く言語：Node.js
