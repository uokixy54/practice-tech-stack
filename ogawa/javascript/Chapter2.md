# javascriptの仕事
    １．データのやり取り
        ・無限スクロール
        
    ２．DOM操作・・・Webページの内容（HTML要素）を動的に変更すること。
        DOM(Document Object Model)とは
            HTMLやXMLでマークアップされた文書を、JavaScriptなどのスクリプトからアクセス、操作できるようにするAPIのこと。
        DOM操作の例
            ・ボタンを押したときに色を変える
            ・要素を追加、削除する etc…
