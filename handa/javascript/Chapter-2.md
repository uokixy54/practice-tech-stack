# 1.javascriptのしごと
    ・データのやり取り
    ・DOM操作
        HTMLのような文書の構造を表現するためのプログラムインターフェースで、「Document Object Model」の略称。
        JavaScriptとの連携により、Webページの内容やスタイルの変更が可能になる。
        HTMLの内容をJavaScriptから操作するためには何らかの橋渡しが必要で、その役割を果たすのがDOM。
        DOMによって、HTMLの各要素をオブジェクトとして扱うことができるようになり、JavaScriptからそのオブジェクトへアクセスして操作できるようになる。
        （参考サイト：https://qiita.com/Shin-mfc/items/38e23da84c7b7a9e5180）

    [バックエンドエンジニアのしごと]
        WebAPIを作ることが仕事な事が多い。
        DBやSaaSなどから情報を取得してフロントが扱いやすいような形にして返すことがバックエンド処理の一つ。
        悪意のあるユーザーからの攻撃を防ぐために認証を挟む事が多い。
    WebAPIがレスポンスとして返すものがJSON。
